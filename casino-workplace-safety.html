Основа безопасности

Корпоративная VPN
для всей команды

Все сотрудники работают через зашифрованный туннель. Независимо от местоположения — офис, дом или командировка — трафик маршрутизируется через защищённую корпоративную сеть.

  • Split tunneling — корпоративный трафик через VPN, личный — напрямую
  • Автоматическое отключение (Kill Switch) при разрыве VPN-соединения
  • WireGuard / OpenVPN протоколы с Perfect Forward Secrecy

Управление доступом

Zero Trust Architecture

Никто не получает доступ по умолчанию — каждый запрос верифицируется независимо от местоположения

Аутентификация

Многофакторная аутентификация

MFA обязательна для всех сотрудников без исключений. Физические ключи безопасности (YubiKey) для администраторов и финансового отдела.

  • TOTP-приложения (Google Authenticator, Authy)
  • Аппаратные ключи YubiKey / FIDO2
  • Биометрия на корпоративных устройствах
Привилегии

Принцип минимальных привилегий

Каждый сотрудник получает ровно столько доступа, сколько нужно для работы — не больше. Роли пересматриваются ежеквартально.

  • RBAC — ролевая модель доступа
  • Привилегированный доступ только через PAM
  • Автоматический отзыв при увольнении — 30 мин
Устройства

MDM — управление устройствами

Mobile Device Management контролирует все корпоративные устройства: принудительное шифрование, удалённое стирание при утере.

  • Jamf (Mac) / Microsoft Intune (Windows/Android)
  • Автопатчинг ОС и ПО без участия пользователя
  • BYOD — изолированный рабочий контейнер
