Что такое LeaderTask Box

LeaderTask Box — коробочная (on-premise) версия таск-менеджера LeaderTask. В отличие от облачной SaaS-версии, сервер разворачивается в вашей инфраструктуре: задачи, файлы, проекты хранятся на вашем сервере, не уходят в чужое облако.

Пошаговая установка
1
Получите дистрибутив и лицензию
Свяжитесь с LeaderTask для покупки Box-версии

Box-версия не скачивается публично — нужно оформить договор и получить дистрибутив от вендора:

  • 📧 Email: 911@leadertask.com
  • 📞 Телефон: +7 4852 68-58-20
  • 🌐 Сайт: leadertask.ru → раздел «LeaderTask Box»

После оплаты вы получаете: установочный дистрибутив + файл лицензионного ключа *.key

2
Подготовьте сервер Windows
Windows Server 2016/2019/2022

Установите необходимые зависимости:

# Откройте PowerShell от имени Администратора

# 1. Установите .NET Framework 4.8
# Скачайте с сайта Microsoft и запустите установщик
# или через Windows Features:
Install-WindowsFeature Net-Framework-45-Features

# 2. Откройте порт 443 в Firewall
New-NetFirewallRule -DisplayName "LeaderTask HTTPS" `
  -Direction Inbound -Protocol TCP `
  -LocalPort 443 -Action Allow
3
Установите MS SQL Server
MS SQL 2016 или новее (Express — бесплатный вариант)

Скачайте и установите SQL Server Express (бесплатно) с сайта Microsoft:

  • При установке выберите режим: Windows Authentication
  • Запомните имя экземпляра (по умолчанию: SQLEXPRESS)
  • Включите TCP/IP в SQL Server Configuration Manager
# Проверьте что SQL Server запущен
Get-Service -Name "MSSQL*"

# Должен быть статус: Running
4
Установите LeaderTask Server
Запустите инсталлятор с правами администратора

Запустите полученный установщик (LeaderTaskServer_setup.exe) от имени Администратора:

  • Укажите папку установки (по умолчанию: C:\LeaderTaskServer\)
  • В поле подключения к БД укажите: localhost\SQLEXPRESS
  • Мастер автоматически создаст базу данных
  • После установки — скопируйте файл лицензии *.key в папку установки
C:\LeaderTaskServer\
├── LeaderTaskServer.exe   ← основной сервис
├── license.key            ← лицензионный ключ (скопировать сюда)
├── config.xml             ← конфигурация
└── data\                  ← данные и файлы
5
Запустите сервис и проверьте
Убедитесь, что сервер работает

LeaderTask Server устанавливается как Windows Service:

# Проверьте статус сервиса
Get-Service -Name "LeaderTaskServer"

# Запустите если остановлен
Start-Service -Name "LeaderTaskServer"

# Настройте автозапуск
Set-Service -Name "LeaderTaskServer" -StartupType Automatic

Откройте браузер и перейдите на:

https://localhost        # локально на сервере
https://ваш-ip-сервера  # из локальной сети
6
Первоначальная настройка
Создание администратора и подключение пользователей

При первом входе в браузере:

  1. Создайте учётную запись администратора
  2. Укажите адрес сервера для подключения клиентов
  3. Раздайте сотрудникам URL или настройте DNS-имя

Подключение мобильных приложений: в настройках приложения указать вместо облака адрес вашего сервера — https://ваш-домен-или-ip

✅
Убедитесь, что все сотрудники подключены к сети (локальной или VPN) при первоначальной активации — это требование лицензирования.

SSL-сертификат и внешний доступ
Опционально — если нужен доступ из интернета

Для работы через интернет (не только LAN):

  • Получите SSL-сертификат (Let's Encrypt или купленный)
  • Настройте в config.xml путь к сертификату
  • Откройте порт 443 на роутере/файерволе
  • Пропишите DNS-запись для вашего домена
# Пример: получить Let's Encrypt через win-acme (Windows)
# 1. Скачать win-acme.exe
# 2. Запустить и указать домен
# 3. Путь к сертификату прописать в config.xml LeaderTask
8
Бэкапы базы данных
Настройте регулярное резервное копирование

Все данные в MS SQL — настройте автоматические бэкапы:

-- SQL: ежедневный бэкап в папку D:\Backups\
BACKUP DATABASE [LeaderTask]
TO DISK = 'D:\Backups\lt_backup_'
  + CONVERT(VARCHAR, GETDATE(), 112) + '.bak'
WITH COMPRESSION, STATS = 10;

Или через Task Scheduler Windows — запускать SQL-скрипт ежедневно в 3:00.

Open-source альтернативы (бесплатно, Docker)

Если вы хотите self-hosted таск-менеджер без лицензионных затрат — эти проекты разворачиваются через Docker Compose за 15 минут:

Проект Сравним с Docker Цена
Vikunja Todoist / LeaderTask ✓ Да Бесплатно
Plane Jira / LeaderTask ✓ Да Бесплатно
Nextcloud Полная экосистема ✓ Да Бесплатно
AppFlowy Notion / LeaderTask ✓ Да Бесплатно
Focalboard Trello / Kanban ✓ Да Бесплатно
⚡ БЫСТРЫЙ СТАРТ — VIKUNJA (15 минут)
mkdir vikunja && cd vikunja

cat > docker-compose.yml << 'EOF'
version: '3'
services:
  db:
    image: mariadb:10
    environment:
      MYSQL_ROOT_PASSWORD: secret
      MYSQL_DATABASE: vikunja
      MYSQL_USER: vikunja
      MYSQL_PASSWORD: secret
    volumes:
      - ./db:/var/lib/mysql

  vikunja:
    image: vikunja/vikunja
    environment:
      VIKUNJA_DATABASE_HOST: db
      VIKUNJA_DATABASE_TYPE: mysql
      VIKUNJA_DATABASE_USER: vikunja
      VIKUNJA_DATABASE_PASSWORD: secret
      VIKUNJA_DATABASE_DATABASE: vikunja
      VIKUNJA_SERVICE_JWTSECRET: your-random-secret-here
    ports:
      - "3456:3456"
    depends_on:
      - db
    volumes:
      - ./files:/app/vikunja/files
EOF

docker-compose up -d
# Открыть: http://localhost:3456
Итог: стоит ли ставить LeaderTask Box?
✓ Подходит если...
  • Команда от 50+ человек
  • Уже используете LeaderTask в облаке
  • Нужен именно LeaderTask UX/функционал
  • Требования по безопасности данных (ФЗ-152, гос)
  • Есть Windows Server в инфраструктуре
✗ Не подходит если...
  • Команда до 20-30 человек
  • Нужен Linux/Docker
  • Ограниченный бюджет
  • Личное использование
  • Нужна гибкость настройки
💡
Рекомендация: Для личного use или небольшой команды — поставьте Vikunja или Plane через Docker. Данные полностью у вас, работает на Linux VPS от 5$/мес, установка 15 минут.
