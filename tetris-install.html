Инструкция

Установка за 5 шагов

1

Скачай IPA файл

Нажми кнопку «Скачать IPA» вверху страницы. Файл называется TetrisApple-signed.ipa (~192 КБ).

2

Установи Sideloadly

Скачай бесплатную программу Sideloadly с официального сайта sideloadly.io. Есть версии для macOS и Windows.

3

Подключи iPhone кабелем

Подключи iPhone к компьютеру USB-кабелем. Если iPhone спросит «Доверять этому компьютеру?» — нажми Доверять.

4

Перетащи IPA и нажми Start

Открой Sideloadly. Перетащи файл TetrisApple-signed.ipa в окно программы. Введи свой Apple ID (обычный, бесплатный) и пароль. Нажми Start — подождёт ~30 секунд.

Если Apple ID использует двухфакторную аутентификацию — Sideloadly попросит код из SMS или уведомления.
5

Разреши запуск на iPhone

На iPhone открой Настройки → Основные → VPN и управление устройством. Найди свой Apple ID в разделе «Разработчик» и нажми Доверять. Готово — запускай Тетрис!

Управление

Как играть

Свайп влево / вправо

Двигай фигуру по горизонтали. Быстрый свайп — быстрое движение.

Свайп вниз

Ускоренное падение фигуры. Зарабатывай бонусные очки за каждую ускоренную клетку.

Тап один раз

Поворот фигуры по часовой стрелке. Работает везде на игровом поле.

Двойной тап

Мгновенный сброс фигуры вниз — максимум очков за drop!

Вопросы

Частые вопросы

Как долго работает приложение?
7 дней с бесплатным Apple ID. После истечения повтори шаги 3–4 (кабель + Sideloadly). Это бесплатно и занимает минуту.
Нужно ли оставлять кабель подключённым?
Нет. Кабель нужен только в момент установки. После того как нажал «Доверять» на iPhone — отключай и играй.
Sideloadly просит пароль от Apple ID. Это безопасно?
Sideloadly известный инструмент для sideloading iOS приложений. Для дополнительной безопасности создай отдельный Apple ID специально для этого — это бесплатно.
Почему нельзя скачать из App Store?
Публикация в App Store стоит $99/год. Это личная игра, и распространяется напрямую как sideload — так же как работают бета-версии приложений у разработчиков.
Работает ли на Android?
Нет — это нативное SwiftUI приложение только для iPhone с iOS 17 и новее.
